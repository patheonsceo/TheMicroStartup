## Overview

Create a simple online form where people can sign up for workshops. This makes it easier to manage registrations and payments than doing everything manually.

---

## Problem Statement

Managing workshop sign-ups through WhatsApp or email gets messy. We need a simple system where people can see available workshops, register, and pay online.

---

## Proposed Solution

Build a basic website page with workshop listings. Add a simple form for each workshop where people can:
- See workshop details (date, time, price, what they'll learn)
- Enter their name, email, phone
- Pay online (using payment link)
- Get confirmation email

Keep it simple - no fancy features needed.

---

## Implementation Plan

1. List all required features (simple list)
2. Build basic webpage (or use simple tool like Google Forms + payment link)
3. Add workshop information
4. Test with a few people
